# BatchProcessAssets

功能

可以批量缩小UE4纹理尺寸，仅用于编辑器下调试，加快纹理资源的载入速度，减少内存占用

提供缩小2-32倍的缩小系数
目前对整个工程所有Texture2D一次性处理

使用方法
1.选择引擎Modes区域下增加的工具栏BatchProcessAssetsEdMode，进入设置页面
2.选择相应的缩放按钮，等待所有处理完成


下一步要优化的内容：
1.部分纹理缩小后不能再次缩小
2.处理过程加入进度显示
3.自定义处理纹理，缩放系数，路径等。
4.更加人性化的界面
5.备份功能

注意事项：
1.处理完成后只会将导入结果存入缓存，需要手动保存
2.保存前自行备份所有纹理数据
3.目前不保证处理后的纹理发布的正确性

所有追求更快加载速度的同学欢迎使用。
